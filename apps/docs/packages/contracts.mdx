# @polis/contracts

Foundry workspace containing the `PolisArchive` smart contract — a multi-tenant registry for onchain community archival.

## PolisArchive.sol

Deployed on **Base mainnet** and **Base Sepolia**.

### Functions

| Function | Access | Description |
|----------|--------|-------------|
| `registerCity(slug)` | Anyone | Register a new city, returns `cityId` |
| `recordSnapshot(cityId, ipfsHash)` | City admin | Record an IPFS content hash |
| `transferAdmin(cityId, newAdmin)` | City admin | Transfer city admin role |
| `getCity(cityId)` | View | Get city info |
| `getLatestSnapshot(cityId)` | View | Get latest snapshot hash |
| `pause()` / `unpause()` | Owner | Emergency controls |

### Events

- `CityRegistered(cityId, slug, admin)`
- `SnapshotRecorded(cityId, ipfsHash, timestamp)`
- `AdminTransferred(cityId, oldAdmin, newAdmin)`

## Development

```bash
# Build
forge build

# Test
forge test -vvv

# Gas report
forge test --gas-report

# Deploy to testnet
forge script script/DeployPolisArchive.s.sol --rpc-url base_sepolia --broadcast
```
