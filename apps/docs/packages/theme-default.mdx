# @polis/theme-default

Provides the baseline dark theme for Polis communities. Pure token export — no JS runtime beyond the object.

## Installation

```bash
pnpm add @polis/theme-default
```

## Usage

```typescript
import themeDefault from '@polis/theme-default';
import { defineConfig } from '@polis/core';

export default defineConfig({
  theme: {
    tokens: themeDefault,
    overrides: {
      primary: '#FF6B35', // Override just what you need
    },
  },
});
```
